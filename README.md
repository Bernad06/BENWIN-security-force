# BENWIN Security Force

## Overview
BENWIN Security Force is a professional manpower and security service platform designed to help businesses, organizations, and individuals quickly connect with trained security personnel and workforce solutions based on their operational requirements.

The platform focuses on providing reliable staffing support, fast service response, and professional workforce management for short-term and long-term needs.

## Services Offered
- Trained Security Guards
- Corporate Security Services
- Event Security Support
- Industrial & Commercial Security
- Housekeeping Staff
- Skilled & Unskilled Manpower
- Immediate Workforce Deployment
- Custom Staffing Solutions

## Key Features
- Professional and modern website design
- Easy service inquiry process
- Quick manpower request support
- Business-oriented service model
- Responsive design for mobile and desktop
- Contact and support information

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose of This Project
This project was developed to create a digital presence for BENWIN Security Force and simplify the process of connecting clients with trusted security and manpower services.
